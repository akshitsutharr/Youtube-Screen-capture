## Install in 3 Easy Steps

### Step 1: Download the Extension
- Download all the files from the `youtube-screenshot-extension` folder
- Keep all files together in one folder

### Step 2: Open Chrome Extensions
- Open Google Chrome
- Type `chrome://extensions/` in the address bar and press Enter
- OR click the three dots menu ‚Üí More Tools ‚Üí Extensions

### Step 3: Load the Extension
- Turn ON "Developer mode" (toggle switch in top-right corner)
- Click "Load unpacked" button
- Select the `youtube-screenshot-extension` folder
- Done! The extension icon will appear in your toolbar

## First Use

1. Go to any YouTube video (example: https://www.youtube.com/watch?v=dQw4w9WgXcQ)
2. Click the extension icon in your Chrome toolbar
3. Set your preferred time range and interval
4. Click "Start Capture"
5. Wait for the ZIP download to complete

## Quick Settings Examples

### Example 1: Quick 1-minute capture
- Start Time: `0:00`
- End Time: `1:00`
- Interval: `5` seconds
- Result: 12 screenshots

### Example 2: Tutorial highlights
- Start Time: `5:30`
- End Time: `15:45`
- Interval: `30` seconds
- Result: ~20 screenshots

### Example 3: Full lecture (2 hours)
- Start Time: `0:00`
- End Time: `2:00:00`
- Interval: `60` seconds
- Result: 2 ZIP files (1 hour each), 120 screenshots total
